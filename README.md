# Desarrollo Web con Bootstrap 5
## Este proyecto está desarrollado con Bootstrap 5 y consiste en un sitio web para una agencia de viajes ficticia llamada "Destinos del mundo". La web incluye un diseño responsive y funcional en dispositivos móviles, tablets y escritorio.
El sitio está compuesto por varias páginas interconectadas: una página principal (index), una sección de destinos y una página de contacto. Además, se ha añadido una carpeta con las imágenes utilizadas y aunque Bootstrap es la base del proyecto, se ha complementado con una hoja de estilos personalizada para mejorar el acabado final. 
En todas las páginas se mantiene una barra de navegación responsive que se adapta automáticamente a dispositivos móviles mediante un menú colapsable. Este Navbar se queda fijo al hacer scroll e incluye el nombre de la agencia, enlaces a las páginas estando resaltada la página activa y un botón que da acceso a un modal interactivo con un formulario de reserva rápida. También hay un footer, estructurado en tres columnas que contiene información corporativa, enlaces rápidos y datos de contacto.
La página principal (index) actúa como presentación de la agencia. Incluye un carrusel destacado con transición automática, controles de navegación y texto sobre las imágenes. A continuación, los destinos se presentan en tarjetas organizadas en un sistema de grid responsive para presentar distintos viajes, incluyendo imágenes, descripciones, precios, valoraciones y etiquetas como “Nuevo” o “Popular”. Además, como ejemplo se ha añadido al primer destino (Tokio) un botón que da acceso a un modal interactivo con un carrusel interno de imágenes, promociones destacadas y un botón que da paso al formulario de reserva rápida. También hay una sección de testimonios de clientes organizada en tarjetas.
La página de “Destinos” amplía la información presentada en la página principal y funciona como un catálogo más detallado de los paquetes turísticos disponibles. Incluye una zona de búsqueda simulada con filtros de destino, continente y precio máximo, diseñada con formularios y selectores. Además, se implementa una tabla comparativa responsive que se adapta en dispositivos pequeños mediante un scroll horizontal y permite visualizar de manera clara la duración de los viajes, el tipo de alojamiento, los servicios incluidos y el precio. También se han añadido badges e iconos visuales para mejorar la legibilidad.
La página de "Contacto" completa la experiencia ofreciendo un espacio de comunicación directa con la agencia. Está estructurada en dos columnas principales: una con un formulario detallado donde el usuario puede enviar consultas personalizadas, y otra con la información corporativa organizada en tarjetas con iconos visuales. También se integra un mapa incrustado mediante iframe y enlaces a redes sociales.
Por otro lado, se ha añadido una hoja de estilos para definir el color de fondo, tipografías, controlar tamaños de imágenes, efectos de transición en tarjetas, personalización de iconos y ajustes específicos para el mapa. Asimismo, se ha incluido una carpeta con las imágenes usadas que se han obtenido de la página Unsplash: https://unsplash.com
En resumen, el proyecto desarrolla la creación de un sitio web completo, funcional y adaptable a diferentes dispositivos utilizando Bootstrap, aplicando un sistema de grid, componentes interactivos, formularios con validación, tablas responsive, modales, iconos y estilos personalizados.
